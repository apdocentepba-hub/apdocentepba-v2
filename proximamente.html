Antes de firmar un acta

Valor alto y baja complejidad. Complementa validador de actas y modelos existentes.

  • Estado: propuesta lista.
  • Requisito: redacción prudente y enlaces a validador.

Listo para prototipo
🧮

Simulador de compatibilidad horaria

Útil antes de APD. Detectaría superposiciones simples sin dictaminar incompatibilidad legal.

  • Estado: prototipo local posible.
  • Requisito: carga manual de horarios.
  • Riesgo: no usar como dictamen.

Generador de mensaje rápido

Mensajes para consultas frecuentes: documentación, licencia, entrevista, reclamo, SAD y registro.

  • Estado: reutilizable desde asistentes actuales.

Criterio para elegir qué sigue

  • Debe resolver un problema frecuente y concreto.
  • Debe poder usarse sin backend cuando sea posible.
  • No debe prometer dictamen legal, salarial o administrativo.
  • Debe enlazar a fuentes oficiales y uso responsable.
  • Debe ser compartible por WhatsApp o grupos docentes.

Qué no necesito para seguir

Puedo seguir mejorando páginas estáticas, redacción, advertencias, navegación, buscador, biblioteca y modelos. No necesito datos externos para eso.

Qué sí necesito para subir las bloqueadas

  • Para calculadora: escalas oficiales por mes y recibos reales anonimizados.
  • Para mapa oficial: padrón normalizado con CUE, nombre, dirección, distrito, latitud y longitud.
