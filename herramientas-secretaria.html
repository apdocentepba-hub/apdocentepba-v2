Secretaría escolar

Herramientas para documentación, constancias y trámites escolares.

Pensadas para escuela secundaria común: recepción de documentación, analíticos parciales, pases, legajos, constancias, observaciones y seguimiento de trámites. No reemplaza normativa ni sistemas oficiales: ordena el trabajo y reduce omisiones.

Fundamento de estas herramientas

📌

Trazabilidad

Todo trámite escolar necesita fecha, responsable, estado y documentación recibida.

Problema real

Cuando no hay registro, después nadie sabe qué falta, quién lo pidió o cuándo se entregó.

✅

Documentación completa

Evita que un trámite avance con datos incompletos o campos vacíos.

Problema real

Analíticos, pases y legajos se traban por DNI, escuela anterior, cursos, plan o constancias faltantes.

🧾

Comunicación formal

Genera textos claros para ventanilla, familias y archivo interno.

Problema real

Un pedido poco claro genera idas y vueltas, discusiones y pérdida de tiempo administrativo.

Analítico parcial

Guía de solicitud / control de analítico parcial

Genera un texto base para pedir documentación y dejar claro qué datos faltan.

Por qué importa

El analítico parcial requiere reconstruir trayectoria: años cursados, escuela anterior, materias aprobadas/pendientes, plan de estudios y datos personales. Si falta algo, el certificado puede salir mal o demorarse.

La guía aparecerá acá.
Checklist

Documentación por trámite

Lista base para ventanilla o revisión interna.

Por qué importa

Permite decirle a la familia/estudiante exactamente qué traer y evita pedidos incompletos.

    Modelo

    Generador de constancia simple

    Texto base para constancia de recepción, trámite iniciado o documentación pendiente.

    Por qué importa

    Cuando se recibe documentación o se informa un faltante, conviene dejar constancia clara: qué se recibió, qué falta y en qué fecha.

    La constancia aparecerá acá.
    Archivo

    Leyendas de observación

    Modelos breves para dejar asentado faltantes, correcciones o situaciones frecuentes.

    Por qué importa

    Las observaciones deben ser claras, objetivas y útiles para quien retome el trámite después.

    Elegí una situación.
    Bitácora

    Seguimiento de trámites de secretaría

    Registro local de trámites pendientes o iniciados. Se guarda en este navegador.

    Por qué importa

    Sirve para controlar qué trámites están pendientes, qué falta y cuándo se actualizó cada caso.
