# Repositorio de materiales

Este espacio organiza materiales institucionales y pedagógicos dentro del mismo repo de APDocentePBA.

## Cómo usarlo
- Entrá a la carpeta que corresponda a tu rol.
- Usá **Add file** en GitHub para subir archivos.
- Tratá de mantener nombres claros y ordenados.

## Estructura inicial
- `01-equipo-directivo`
- `02-secretaria`
- `03-profesores`
- `04-preceptoria`
- `05-eoe-orientacion`
- `06-ematp-tecnicos`

## Nota
La subida directa desde el panel todavía no está activada. En esta primera versión, el panel funciona como lanzador ordenado hacia carpetas reales del repo.
